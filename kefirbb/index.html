KefirBB

About

KefirBB is the open source Java text processor. Initially it was developed as bbcode to html converter. KefirBB has flexible configuration in XML-file. Developer can define specified codes with it. This futures allows use KefirBB as any text processor you like. For example: XML2XML, XML2HTML, XML2JSON, JSON2XML and other.

Maven

<dependency>
    <groupId>org.kefirsf</groupId>
    <artifactId>kefirbb</artifactId>
    <version>1.0</version>
</dependency>
